# Hybrid-Agent 暗色模式规格文档

## 1. 项目概述

- **项目名称**: Hybrid-Agent RAG 前端
- **项目类型**: Streamlit Web 应用
- **核心功能**: 为 RAG 知识库聊天应用添加暗色/亮色模式切换功能
- **目标用户**: 使用 RAG 知识库和 AI 助手的开发者/用户

## 2. UI/UX 规格

### 2.1 布局结构

- **页面结构**: Streamlit 侧边栏 + 主聊天区域
- **切换按钮位置**: 侧边栏顶部，标题旁边
- **响应式**: 桌面端和移动端自适应

### 2.2 视觉设计

#### 亮色模式 (Light Mode) - 当前配色
```css
--bg-primary: #FFFBF7       /* 主背景 - 暖白色 */
--bg-secondary: #FFF5E6     /* 次要背景 - 浅橙色 */
--bg-sidebar: #FFFFFF        /* 侧边栏背景 */
--accent-primary: #FF6B35    /* 主强调色 - 橙色 */
--accent-secondary: #FF8C42 /* 次要强调色 */
--text-primary: #1A1A1A     /* 主文字 */
--text-secondary: #6B7280   /* 次要文字 */
--text-placeholder: #9CA3AF /* 占位符文字 */
--border: #E5E7EB           /* 边框颜色 */
--shadow: rgba(0, 0, 0, 0.08) /* 阴影 */
```

#### 暗色模式 (Dark Mode) - 新配色
```css
--bg-primary: #0D1117       /* 主背景 - 深蓝黑 */
--bg-secondary: #161B22     /* 次要背景 - 深灰 */
--bg-sidebar: #0D1117       /* 侧边栏背景 */
--accent-primary: #FF6B35   /* 主强调色 - 保留橙色 */
--accent-secondary: #FF8C42 /* 次要强调色 */
--text-primary: #E6EDF3     /* 主文字 - 浅灰白 */
--text-secondary: #8B949E   /* 次要文字 */
--text-placeholder: #6E7681 /* 占位符文字 */
--border: #30363D            /* 边框颜色 */
--shadow: rgba(0, 0, 0, 0.4) /* 阴影 */
```

### 2.3 组件规格

#### 主题切换按钮
- **位置**: 侧边栏顶部右侧
- **图标**: 🌙 (暗色) / ☀️ (亮色)
- **样式**: 图标按钮，hover 时有背景高亮
- **尺寸**: 36x36px
- **动画**: 150ms 过渡效果

#### 消息气泡
- **用户消息**: 暗色模式下使用 `#21262D` 背景
- **AI 消息**: 暗色模式下使用 `#161B22` 背景，带 `#30363D` 边框

#### 输入框
- **背景**: 暗色模式下使用 `#0D1117`
- **边框**: 暗色模式下使用 `#30363D`
- **文字**: 暗色模式下使用 `#E6EDF3`

#### 文件上传区域
- **背景渐变**: 暗色模式下使用 `linear-gradient(135deg, #161B22 0%, #0D1117 100%)`
- **边框**: 暗色模式下使用 `#30363D`

### 2.4 动画效果
- 主题切换: 200ms 渐变过渡
- 按钮 hover: 150ms 缩放效果
- 消息出现: 200ms 淡入效果

## 3. 功能规格

### 3.1 核心功能
1. **主题切换**: 点击按钮在亮色/暗色模式间切换
2. **状态持久化**: 主题设置保存在 session_state 中（页面刷新后重置为默认）
3. **默认主题**: 首次加载默认为亮色模式

### 3.2 用户交互流程
1. 用户点击侧边栏顶部的主题切换按钮
2. 主题立即切换（无页面刷新）
3. 所有 UI 元素同步更新配色

### 3.3 技术实现
- 使用 Streamlit session_state 存储主题状态
- 使用 Streamlit 的 CSS 注入机制 (`st.markdown` + `unsafe_allow_html=True`)
- 通过 CSS 变量实现主题切换

## 4. 验收标准

### 视觉检查点
- [ ] 暗色模式下主背景为深蓝黑色 (#0D1117)
- [ ] 暗色模式下文字清晰可读，对比度足够
- [ ] 主题切换按钮在两种模式下都可见且可点击
- [ ] 消息气泡、输入框、侧边栏在暗色模式下样式正确

### 功能检查点
- [ ] 点击切换按钮主题正确切换
- [ ] 切换后所有 UI 元素同步更新
- [ ] 页面内发送消息后主题保持不变

### 兼容性
- [ ] 在 Chrome、Safari、Firefox 中正常工作
- [ ] 暗色模式不破坏现有功能
