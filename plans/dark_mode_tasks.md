# Hybrid-Agent 暗色模式实现任务清单

## 任务概览
为 Hybrid-Agent RAG 前端应用添加暗色模式支持，包含主题切换按钮。

## 实现步骤

### Phase 1: 基础设置
- [ ] **T1.1** 在 session_state 中添加 theme 状态（默认: "light"）
- [ ] **T1.2** 定义亮色和暗色模式的配色方案变量

### Phase 2: CSS 样式更新
- [ ] **T2.1** 更新基础 CSS 变量，使用 CSS 变量替代硬编码颜色
- [ ] **T2.2** 添加暗色模式专属 CSS 样式
- [ ] **T2.3** 添加主题切换按钮的 CSS 样式

### Phase 3: 主题切换按钮
- [ ] **T3.1** 在侧边栏顶部添加主题切换按钮
- [ ] **T3.2** 实现按钮点击事件处理
- [ ] **T3.3** 添加按钮图标动态切换（🌙/☀️）

### Phase 4: 组件暗色样式适配
- [ ] **T4.1** 更新消息气泡暗色样式
- [ ] **T4.2** 更新输入框暗色样式
- [ ] **T4.3** 更新文件上传区域暗色样式
- [ ] **T4.4** 更新侧边栏暗色样式
- [ ] **T4.5** 更新统计卡片暗色样式
- [ ] **T4.6** 更新工具审批面板暗色样式
- [ ] **T4.7** 更新思考过程面板暗色样式

## 验收清单
- [ ] 主题切换按钮可见且可点击
- [ ] 暗色模式下所有组件颜色正确
- [ ] 亮色模式保持原有设计
- [ ] 主题状态在页面交互中保持
